# Custom AI Study Assistant & Curriculum Engine

A production-ready Python application leveraging the official Google GenAI SDK to orchestrate dynamic model parameters, system prompts, and personas, served via an interactive web interface using Gradio and permanently deployed to the cloud.

## 🚀 Features
- **Official SDK Engineering:** Clean backend integration using native programmatic client variables (`genai.Client`).
- **Dynamic Persona Ingestion:** System instruction matrix mapping UI choices to strict structural context rules.
- **Deterministic Token Safety:** Managed temperature boundaries and max output controls inside generation configs.
- **Gradio Serving Layer:** Instantly packages application functionality into an interactive interface layout.

## 🛠️ Tech Stack
- **Language Core:** Python 3.10+
- **Inference Model:** Google Gemini API (`gemini-2.5-flash`)
- **UI Framework:** Gradio Engine
- **Hosting Targets:** Hugging Face Spaces
